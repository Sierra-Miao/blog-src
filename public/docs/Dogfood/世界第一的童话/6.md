# 六、假车票居然能用？
&emsp;&emsp;“来自手长星的两位朋友们，我叫芜湖，我这有一张宇宙飞船的VIP车票，只要你……我就可以把这张车票给你！”好耶和起飞半信半疑，“你该不会是电信诈骗吧？是不是等我做完之后还要打各种钱，最后还是假的？”芜湖不紧不忙地回答：“那要是我还要你做别的东西你就弃坑咯，反正我要你做的也不是什么坏事，你也不看看水獭们都胖成球了，这票要是现在不要以后就没机会了嗷！”好耶和起飞这下是完全信了，想到马上就可以去地球吃到朝思暮想的香菜牛肉了，连忙答应，欢呼雀跃。

&emsp;&emsp;在那个月黑风高的夜晚，水獭们已经准备好了明天的早餐，安稳地睡去。好耶和起飞爬进厨房，往白粥里面加了点华莱士调味剂，心中默想，不仅可以给自己拿到车票，还能给水獭星寡淡无味的伙食增加一点风味，让水獭们不再胖成一个球，真是一举多得还能做大善人呀。

&emsp;&emsp;第二天早上，水獭们明显感觉到了白粥的美味，不再是往常那种寡淡无味，他们认为是手长星的两位新朋友给他们带来了如此的幸运。过了不久，水獭们排泄的欲望愈加强烈，仅仅用了半天时间，水獭的球化就消失了，水獭们也第一次体会到自己排泄物的奶香居然能对整个星球的气氛产生如此美妙的点缀效果。

&emsp;&emsp;“冲鸭！把水獭星拿下！”失去了球化的水獭们已经不能再以滚滚乐击退米老鼠了，只能被激光武器挠痒痒挠得笑不拢嘴。在又无奈又欢乐的气氛下，水獭们答应要把票投给米老鼠。

&emsp;&emsp;就在此时，还在呼呼大睡的起飞和好耶突然被从天而降的金块砸醒。“哎呀，疼疼疼！”“快看！我们金光闪闪的VIP车票！我们可以去地球啦！”“芜湖，永远滴神！”

&emsp;&emsp;此时米奇正从自己的飞船走下来，正准备载起飞和好耶到宇宙交通局乘坐宇宙飞船。

&emsp;&emsp;他们很快就到了宇宙交通局驻哺乳区分局了。验票员接过好耶和起飞手里的票，瞬间就皱紧了眉头，低声交谈，“这就是很久没有人买过的VIP车票吗，天呐不会又改了款式了吧？”“我也不知道呀，谁叫我们上次开会的时候都在摸鱼……”“天呐，这么豪华的样子不会是传说中的隐藏款吧，搞不好是个大客户，可不能怠慢！”

&emsp;&emsp;“有什么问题吗？”“没，没什么问题，你们手里的票是等级最高的，来这边请，这架宇宙飞船你们随便开。”

&emsp;&emsp;“好耶！”好耶和起飞的臂高兴得转了起来，跟接了生物电源一样。（完）
